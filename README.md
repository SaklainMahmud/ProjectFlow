# ProjectFlow

ProjectFlow is a full-stack project management application built with React and Node.js.

## Structure

projectflow/
├── frontend/ # React (Vite)
└── backend/ # Node.js + Express + MongoDB

## Features
- User authentication (JWT)
- Project & task management
- Task status workflow
- Protected routes
- Clean UI

## Tech Stack
Frontend:
- React, Vite, Axios

Backend:
- Node.js, Express, MongoDB, JWT

## Setup

### Backend
```bash
cd backend
npm install
npm run dev
